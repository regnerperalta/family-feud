Instructions ×
Setup:
  • You need to set your monitor to this resolution: 1024 x 768
  • Load the page
  • Go into setup (Press 'S')
  • Update the team names (if desired)
  • Upload the questions file (see example)
  • VERY IMPORTANT: All question files must follow the format specified in the example
Key controls:
  • S - Setup (open/close)
  • ? - Help (open/close)
  • W - Play theme song
  • N - New Game
  • Q - Next Question
  • Y - Display/Hide Question
  • 1 .. 8 - Reveal the appropriate answer
  • X - Strike
  • A - Team 1 Wins
  • B - Team 2 Wins
  • R - Reset the Number of Strikes to 0
  • - - Reduce Number of Strikes (when done by accident)
  • T - Start timer
  • [ - Open Bracket
  • ] - Close bracket
  • O - Toggle All Lights On/Off
  • Z - Blink
  • U - Bottom to Top Light Effect
  • L - Left to Right Light Effect
  • R - Right to Left Light Effect
  • D - Top to Bottom Light Effect
